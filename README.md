# Проект команды Don't Stop
### 

## Установка / Использование 
### Скачайте и установите зависимости
Выполните в консоли:
```
https://github.com/DaryaLugmanova/leadersofdigital.git
cd leadersofdigital
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
```
